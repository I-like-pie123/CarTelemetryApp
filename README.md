# CarTelemetryApp
A robust Android application built with Kotlin using Jetpack Compose, designed to interface with OBD-II sensors via Bluetooth connectivity. This application is designed to calculate real time driving performace, provide a live UI with various parameters, and securely sync data to Google Sheets and ServiceNow. The data is sent both live, includng parameters like RPM, Coolant temp, and throttle position, but a final summary of the drive is sent as well with some of the data. For security, Secrets Gradle Plugin is used, to ensure sensitive information like the ServiceNow instance and password remains out of the repository. The full process involves the app connecting via Bluetooth to an OBD-II device and then it begins pulling data from it, as well as from the phone GPS. It also pushes that data every 5 seconds to a Google Sheet and to ServiceNow. The app is also designed to auto pause after it detects a minute of stopping and resumes once the vehicle is moving again. Finally, when the trip is complete it sends summary data to ServiceNow and Sheets and then sends some of that trip summary data to you, and stores it in Trip History which can be accessed with the Trip History button.
